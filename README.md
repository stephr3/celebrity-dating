A fun quiz to find out which celebrity you should date.

* CSS<br>
* Bootstrap<br>

Installation
------------
Clone Github Repository:
```
$ git clone https://github.com/stephr3/celebrity-dating.git
```
Open index.html file in your browser of choice.
